🛡️ Troubleshooting Common Errors

Sometimes the installer fails on Windows 11. These tips solve most installation blocks.

Installation Access Denied

Right-click the downloaded file. Select "Run as administrator" from the menu. This grants additional permissions needed for network monitoring tools.

Driver Conflicts

NetWorx installs a small driver to watch your traffic. If the installer hangs, ensure no other network traffic monitors run in the background. Uninstall old versions of similar tools before you attempt a new install.

Antivirus Interference

Security software sometimes flags network tools. If your firewall or antivirus stops the process, check the protection history. Add an exception for the installer if you trust the source.

🚀 Using NetWorx

Once you install the tool, you will see a small icon in your system tray. Right-click this icon to see your choices. You can open graphs to visualize usage. The settings menu allows you to set alerts if your data usage hits a limit.
